## 0. RDBMS 

Maria DB

- RDBMS를  쓰는 이유 `중복 방지, 효율성 향상`

  1) 업무파악(Data)

  2) DB 모델링 (테이블 중심)

  3) 테이블 (제약사항, 관계, 주요키, 외래키 등) 

  4) 데이터 입력 

  5) 데이터 조회, 수정, 삭제

  6) 관리(백업, 보안 등)
